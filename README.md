# 问卦 · 周易占筮

一个精致的《周易》占筮网站：完整收录通行本《周易》经传全文，以传统占法起卦，由智谱 GLM 大模型依经据典解卦。

## 特点

- **经文完备**：六十四卦卦辞、爻辞、彖传、象传（大象与小象）、乾坤文言、用九用六，以及《系辞》《说卦》《序卦》《杂卦》全文，可在「卦典」「易传」页查阅。
- **占法标准**：
  - *三钱法*（火珠林法）：三枚铜钱掷六次，背 3 字 2，六为老阴、七少阳、八少阴、九老阳；
  - *大衍筮法*（《系辞》古法）：真实模拟「分二、挂一、揲四、归奇」，四营十八变而成卦；
  - 随机数取自 `crypto.getRandomValues`（加密级随机）。
- **变占有据**：依朱熹《易学启蒙》变占法则自动标注「当占之辞」（无变爻至六爻皆变共七则）。
- **AI 解卦**：将所问之事、六爻记录、本卦之卦经传原文与变占法则一并呈给 GLM，流式输出解读。
- **零依赖**：Node.js 原生实现，无任何 npm 依赖；前端原生 ES Modules，无构建步骤。

## 快速开始

需要 Node.js ≥ 18。

```bash
cd wengua
copy .env.example .env      # 填入 GLM_API_KEY（也可以稍后在页面「设置」中填）
node server.js
```

打开 http://localhost:3000 即可。

## 配置

| 变量 | 说明 | 默认 |
| --- | --- | --- |
| `GLM_API_KEY` | 智谱开放平台 API Key（[open.bigmodel.cn](https://open.bigmodel.cn)） | 空 |
| `GLM_MODEL` | 模型名 | `glm-5.2` |
| `GLM_BASE_URL` | API 地址 | `https://open.bigmodel.cn/api/paas/v4` |
| `PORT` | 端口 | `3000` |

API Key 也可以不放服务端：直接在网页「设置」里填写，密钥保存在浏览器 localStorage，由本地服务转发调用。

## 项目结构

```
wengua/
├─ server.js              # 零依赖 HTTP 服务：静态托管 + GLM SSE 代理
├─ scripts/build-data.js  # 由 data/raw 原始文本生成前端数据（含校验与勘误）
├─ data/raw/              # 《周易》经传原始数据（来自开源项目 open-iching，已勘误）
└─ public/
   ├─ index.html
   ├─ css/style.css       # 设计系统：宣纸 · 墨 · 朱砂，含暗色「墨夜」主题
   └─ js/
      ├─ app.js           # 视图与交互
      ├─ divination.js    # 起卦（三钱 / 大衍）、变卦、朱熹变占法则
      ├─ prompt.js        # AI 解卦提示词
      ├─ api.js           # 设置存取与 SSE 流式调用
      ├─ markdown.js      # 轻量 Markdown 渲染
      └─ data/            # 生成的经传数据（hexagrams.js / texts.js）
```

## 文本来源与勘误

经传文本取自开源项目 [open-iching](https://github.com/john-walks-slow/open-iching)，并以**通行本/王弼本**为准做过逐字校勘：卦辞（64）、爻辞（386，含用九用六）、大象传（64）已全部逐字核对；《序卦》《杂卦》《说卦》《系辞上》《文言》已对照中国哲学书电子化计划（ctext.org）与维基文库逐字比对。构建脚本 `fix()` 中集中订正的讹误/脱字：

| 位置 | 原 | 正 |
| --- | --- | --- |
| 乾九二爻辞·小象 | 见龙**再**田 | 见龙**在**田 |
| 系辞上（4 处） | **懮**（虞/悔吝/不懮/同懮） | **忧**（憂） |
| 系辞上 | 效法之**为**坤 | 效法之**谓**坤 |
| 系辞上 | 以君子将有为也 | **是**以君子将有为也 |
| 系辞上 | 先号**啕** | 先号**咷** |
| 系辞下 | **履**校灭趾 | **屦**校灭趾 |
| 系辞·大有上九 | 自天**佑**之 | 自天**祐**之 |
| 说卦·离 | 为甲**胃** | 为甲**胄** |
| 文言·乾 | 君子行此四**者** | 君子行此四**德**者 |
| 恒卦彖传 | （源数据缺失） | 已补录 |

> 尚未逐字联网通校的部分：《系辞下》第二章及以后、小象传（384 条爻象）、彖传全文——已做知识性抽查未见明显问题，但严肃引用前建议再跑一次逐字比对。校勘脚本见 `scripts/`（另附 `test-randomness.js` 起卦概率检验）。

如需重新生成数据：`node scripts/build-data.js`。
